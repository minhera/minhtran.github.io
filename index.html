WordFunctionLanguage

A lightweight language for text and file processing (.wfl files)

Purpose

Word Function Language (WFL) is a beginner-friendly scripting language focused on string operations, word analysis, and text file processing.
No mathematics, no syntax complications.
As well as assistive tool for Scrabble and Wordle.

Language Features

  • string s = "hello world" – store a string
  • file f = "sample.txt" – read a file into memory
  • save s to "out.txt" – write a string to a file
  • s – print string
  • cword s – count words
  • csent f – count sentences
  • cchar s – count characters
  • used s – list of used characters
  • count "the" f – count specific terms
  • unscramble "meat" – find valid English words
  • wordle "meat" withlength 4 pattern _e__ – match words fitting a pattern
  • removeword "hello" s – delete a word from string
  • for sentence in f – loop through each sentence in a file

Sample 1: Count Characters

string text = "Hello world!"
cchar text
count "o" text
12 characters
2 o in "Hello World"

Sample 2: Wordle Pattern Match

wordle "meat" withlength 4 pattern _e__
team
mean
meat

Sample 3: Sentence Rules

file story = "story.txt"
rule lessthan 5 print "short"
rule morethan 10 print "long"
for sentence in story
short
A normal sentence in story file.
long
