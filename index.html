Word Function Language (.wfl)

About the Language

Purpose: WFL is designed for simple and readable string manipulation, ideal for text processing, language puzzles, and basic linguistic logic without numeric complexity.

Philosophy: Use natural, human-like syntax for defining strings, analyzing them, applying rules, and extracting meaningful patterns or transformations.

Key Features:

  • Simple assignments with string and file
  • Character, word, and sentence counters
  • Wordle-style pattern matching with withlength, exclude, and pattern
  • Unscramble feature that finds real words from jumbled letters
  • Rule-based sentence analysis using rule and for sentence in
  • Word/letter counting with count
  • Word removal using removeword
  • Output generation with user-defined logic

Sample Programs

Program 1: Count Characters

string text = "Hello world!"
cchar text
12 characters

Program 2: Wordle with Pattern

wordle "meat" withlength 4 pattern _e__
team
mean
meat

Program 3: Rule-Based Sentence Analysis

file story = "story.txt"
rule lessthan 5 print "Too short"
rule morethan 10 print "Too long"
for sentence in story
Too short
A good sentence.
Too long
